ESP32-S2 Stooged 2 Minute Install



A version of the ESP32-S2 Server 9.00 code created by Stooged.
Inspired and created after watching this video by Brian Lough.
Modified image contains:

  • No Password for the PS4_WEB_AP Access Point
  • USB Wait Time is now at 3 seconds vs 10 seconds


    • Note: Payloads are not included. They can be added by connecting to this device via WiFi and navigating to the admin.html web interface.

    Install





    NOTE: Close the Arduino IDE or anything that uses the port or this won't work!
    Your browser does not support the Web Serial API. Try the latest Chrome browser (Version 98.0.4758.102 (Official Build) (64-bit) is what I used)
