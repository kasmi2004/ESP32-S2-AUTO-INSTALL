ESP32-S2 Web Installer for PS4 Homebrew



A web installer for a modified version of the ESP32-S2 Server 9.00 created by Stooged.

I learned about the web installer method after watching this video by Brian Lough.

I've modified the base image with the following:

  • GoldHEN v2.2 Added
  • No Password for the PS4_WEB_AP Access Point
  • USB Wait Time is now at 3 seconds vs 10 seconds

    • Note: Payloads are not included in this release. They can be added by connecting to this device via WiFi and navigating to the admin.html web interface.

    Install





    Note: Close the Arduino IDE or anything that uses the COM port or this won't work!

    Your browser does not support the Web Serial API. Try the latest Chrome browser (Version 98.0.4758.102 (Official Build) (64-bit) is what I used)
